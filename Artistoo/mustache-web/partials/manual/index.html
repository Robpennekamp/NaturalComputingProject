This manual contains detailed tutorials on how to use Artistoo to build your simulations. We are still adding to it, so stay tuned for the most recent version! If you have questions or would like to request a tutorial on some feature, get in touch using the discussion page of our GitHub repository.

If you prefer a more hands on approach, you can also learn to use the framework by playing with the examples (with provided code). For detailed documentation of all classes and methods, see the full documentation. A guide for Artistoo developers will be made available at some point in the future, but until then, you can use the documentation to get started and the discussion page on GitHub to ask questions.
